# Gender Reveal Live Voting Platform

## Resumen
Aplicación web para un evento de Gender Reveal.

Los invitados escanearán un código QR que los dirigirá a un sitio web donde podrán:
- Registrar su nombre.
- Votar si creen que el bebé será Niño o Niña.
- Dejar un mensaje para los papás.

Los resultados se visualizarán en tiempo real en una pantalla principal proyectada durante el evento.
